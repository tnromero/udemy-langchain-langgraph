# Projeto de estudos do curso "LangChain e LangGraph: Crie Agentes de IA com LLMs e RAGs"

Curso disponível em: https://www.udemy.com/course/lanchain/

## Setup

1. Altere o arquivo config.yaml com o seu token
2. Crie uma Virtual Env (venv) do Python
3. Ative a venv
4. Instale as dependências
    ```shell
    pip install -r requirements.txt
    ```
